# House rules

Edit this freely as your table forms preferences. Rulings made at the table go
under "Active" and are then used consistently. Quick toggles (rules
strictness, beginner mode, public rolls, narration style, free-text rules)
live in the web companion's Settings tab (state/settings.json) — this file is
for rulings that deserve prose.

## Active

- **Real dice only.** All rolls — including Director "secret" rolls — go through `tools/dice.py`. The DM never says a number without a roll output to point at. (ON by default — it's the whole point of this project.)
