# Skill checks — when and how

## When to call for a roll

Only call for a check when **all three** are true:

1. There is a meaningful chance of failure.
2. Failure is interesting (it costs time, alerts an enemy, gives a worse version of what they wanted — not "nothing happens").
3. The PC could plausibly succeed.

If the action is trivial, narrate success. If it's impossible, narrate failure. Don't roll for either.

## Setting a DC

Pick from the DC table in `srd-reference.md`. Default to 15 for "an experienced adult would succeed about half the time." Adjust by ±5 for circumstances. Don't roll a die to set the DC.

## Advantage / disadvantage triggers

- **Advantage** when conditions strongly help (perfect tool, ally distracting, plenty of time, environmental aid).
- **Disadvantage** when conditions strongly hurt (improvised tool, time pressure, distraction, partial restraint).
- Sources don't stack — you're "with" or "without." A single source of each cancels.
- Help action grants advantage on one check (Help only works if the helper could plausibly attempt the task themselves).

## Group checks

When the whole party tries together: each rolls; if **at least half succeed**, the group succeeds. Use when stealth or travel speed depends on everyone.

## Contested checks

Two creatures roll the same or related checks; higher wins. Ties = nothing happens (deferring to status quo / defender).

## Passive checks

Use when:
- You don't want to telegraph there's something to find ("you spot the trap" without asking for a roll).
- The PC is doing something repeatedly over a long period (long-distance travel, watch).

Passive = 10 + all relevant mods. Advantage = +5, disadvantage = -5.

## Take 10 / Take 20

5e doesn't use Take-10/Take-20 by default. Use passive checks for the take-10 use case. For take-20 ("they have all day, no consequences"), narrate success.

## Common skill associations

| Skill | Typical uses |
|-------|--------------|
| Acrobatics | Balance, tumble, escape grapple (Dex option) |
| Animal Handling | Calm a beast, ride a difficult mount |
| Arcana | Recall lore about spells, magic items, planes |
| Athletics | Climb, jump, swim, grapple, shove |
| Deception | Lie with a straight face |
| History | Recall historical / political / war lore |
| Insight | Read intent, detect lie (vs Deception) |
| Intimidation | Coerce through threat |
| Investigation | Search, deduce, examine clues |
| Medicine | Stabilize dying, diagnose |
| Nature | Recall lore about flora, fauna, weather, terrain |
| Perception | Notice things in the moment |
| Performance | Entertain, focus attention |
| Persuasion | Convince in good faith |
| Religion | Recall deities, rites, holy symbols |
| Sleight of Hand | Pickpocket, palm an item |
| Stealth | Hide, move silently |
| Survival | Track, navigate, forage |

## What rolls do NOT do

- They don't override stated character knowledge. If a PC obviously can't know something (they weren't there, no exposure), no Arcana / History check unlocks it.
- They don't bypass roleplay. NPCs have motivations; Persuasion sets the ceiling, not the floor. A check can't make a parent betray their child for 2 gp.
- They don't replace clever player thinking. Reward described action over "I roll Perception."
